# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project — issues, pull requests, discussions, code review, or any other channel — you agree to abide by its terms.

## Reporting

If you experience or witness behaviour that violates the Code of Conduct, please report it privately to **jpaulo4799santos@gmail.com**. Reports will be handled confidentially. The maintainer will respond within 5 business days and will follow the [Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines) defined in the Covenant when deciding on consequences.

For security vulnerabilities, see [SECURITY.md](SECURITY.md) instead — a separate channel.
